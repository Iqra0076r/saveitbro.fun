## 📞 Technical Support

**Dev Server Issues:**
1. Kill all node processes: `Get-Process node | Stop-Process -Force`
2. Clear cache: `rm -rf node_modules dist .vite`
3. Reinstall: `npm install`
4. Restart: `npm run dev`

**Build Issues:**
1. Check TypeScript: `npm run lint`
2. Verify dependencies: `npm list`
3. Check Node version: `node --version` (should be 16+)

**Backend Connection:**
1. Verify Flask is running: `curl http://localhost:8000/health`
2. Check `.env` API URL
3. Review browser console for errors
